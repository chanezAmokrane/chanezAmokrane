## ✦ Ce qui me différencie

> La plupart des profils data savent **analyser**.  
> La plupart des profils dev savent **construire**.  
> Moi, je fais les deux — et je sais **expliquer pourquoi ça compte**.

Je suis à l'intersection de trois mondes :

```
[ Développement ]  ──────  [ Data & BI ]  ──────  [ Compréhension métier ]
       ↑                         ↑                          ↑
  Je construis             Je transforme               Je restitue
  les systèmes             les données                 les insights
```

Ce n'est pas courant. Et c'est précisément ce qui me permet d'aller plus loin qu'un dashboard.

## 🧬 Mon approche en 3 étapes

```
① COMPRENDRE        →    Je pose les bonnes questions avant d'ouvrir un fichier.
② CONSTRUIRE        →    Je modélise proprement avant de faire un visuel.
③ RESTITUER         →    Je transforme un résultat technique en insight actionnable.
```
